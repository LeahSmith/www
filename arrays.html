Arrays

What is an array?

An array is a collection of variables.
Arrays have three important properties:
1. represent a group of related data
2. all data within an array have the same type
3. size of an array is fixed once it is created

Declaring an Array

You need to give the array a name and specify its type:
int myFirstArray[];
No memory is allocated for the array here (can’t as it doesn’t know how big the array is).

To give the array a size (and allocate memory):
myFirstArray = new int [5];
This specifies that the number of elements (items) in the array is 5

We can also declare and allocate memory in one line:
int myFirstArray[] = new int [5];

You can use a constant to set the size of the array
final int MAX = 10;
int array2[] = new int [ MAX ]; // [] in different position

Inititalising an Array

int myFirstArray[] = { 1, 1, 1, 1, 1 }; // declare and initialise together

// or initialise afterwards.
int myFirstArray [] = new int [ 5 ];
for (int i = 0 ; i <= 4 ; i++ ) { myFirstArray[ i ] = 1 ; }

Array Indexing

In Java, an arrays’ elements start out at index 0
For example: int array [] = {18, 16, 14,12, 10, 8};

Index Value
0 18
1 16
2 14
3 12
4 10
5 8

Can change the value in an array easily:
array[5] = 9;
Can get the value stored in the array easily:
System.out.println ("The value at the fifth position is " + array[ 4 ] );

Random Numbers

There are a number of ways to generate random numbers.
We will look at generating random integers:
We need to import the following:
import java.util.Random;
And then to do the following:
Random randomGenerator = new Random();

randomInt = randomGenerator.nextInt(1000);
Will generate a random number between 0 and 1000

for(int i = 0; i < array.length; i++)
{
randomInt = randomGenerator.nextInt(1000);
array[i] = randomInt;
} // end for

Arrays and Loops

Loop can be used to print the contents of an array

int array []= new int [ 5 ];
for (int i = 0 ; i < array.length; i++ )
{
System.out.println (“Array element " + i + " is equal to " + array[i]);
} // end for

If you try to access a value beyond the size of the array you will generate an out of bounds error
