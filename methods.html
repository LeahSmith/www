Methods

To date we have seen a number of methods in Java without really knowing what they were:
public static void main(String args[])
You have used methods with Strings e.g.
.charAt(), .indexOf(), .toUpperCase();
A method can return a value e.g. charAt returns a character
If a method does not return a value, its return type is void.
A method can take 0 or more parameters.

Calling Methods

In Java, a method is a piece of code that has been given a name.
To call a method you need: An object, The method name, Some parameters (0 or more)
Example:
object.methodName(parameter1, parameter2)
scan.nextLine();
Very often a method will return a value of some type
s1.charAt(5);

Method Signature

The method signature tells us a lot – the name of the method, the input parameters and the return type
For Example:
public static void multiply(int x, int y)

Return Type Method Name Input Parameters
public static void multiply (int x, int y)
How to call the Method

To call the (use the code within the) method
public static void multiply(int x, int y)
We do:
multiply(5,7);
